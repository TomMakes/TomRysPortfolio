Dangers of the Road

This is a data visualization that showcases the rates of fatal collisions per billion miles traveled in each state.
The goal of the visualization is to give the user a quick look at which states are safer to drive in than others.
There is an optional interactive story which guides the user through navigating the visualization, as well as tells them a little about the states.
It uses the D3.js library to draw the graphs.

Sticker Trader

For this project I programmed a website which allows users to open sticker packs to obtain tradable stickers.
The user can view their collection of stickers, and trade individual stickers for credits.
I used a MongoDB database to store the sticker data for each user.
A node server holds all the data for the users so that they can have individual accounts saved online that are protected.

Fun Time Fuel

For this project I created a website which has two APIs interact with each other.
This website allows the user to pair a event with nearby restaurant choices.
I used the Eventful API as well as Yelp API.
In order to make the Yelp API work, I have a node server that sends the users request to Yelp, then sends data back to the webpage.
